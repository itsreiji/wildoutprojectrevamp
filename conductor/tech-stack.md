# Technology Stack - WildOut!

## Frontend
- **Framework:** React 19 (TypeScript)
- **Build Tool:** Vite 7.2
- **State Management:** TanStack Query (v5) for server state; React Context API for global UI state.
- **Routing:** React Router Dom (v7)

## Styling & UI
- **CSS Framework:** Tailwind CSS 4.1
- **Component Library:** Shadcn UI (Radix UI primitives)
- **Icons:** Lucide React, React Icons
- **Animations:** Motion (Framer Motion)

## Backend & Infrastructure
- **BaaS:** Supabase (PostgreSQL database, Authentication, File Storage)
- **API Framework:** Hono
- **Workflow Orchestration:** Inngest (for background jobs and automation)

## Data Handling & Validation
- **Schema Validation:** Zod
- **Forms:** React Hook Form with Zod resolver

## Testing & Quality
- **Test Runner:** Vitest
- **Testing Library:** React Testing Library
- **Task Management:** Turbo (Monorepo/Task orchestration)
